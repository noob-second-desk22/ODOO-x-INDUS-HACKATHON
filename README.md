# ODOO x INDUS HACKATHON

This is a simple inventory management system built for the ODOO x INDUS hackathon. It helps businesses keep track of their stock, manage receipts and deliveries, and see their movement history in one place.

### Tech Stack & Frameworks
We used these tools to build the project:
- **Python:** The main programming language.
- **Flask:** The web framework that runs the site.
- **SQLModel:** Used to handle the database (it's a mix of SQLAlchemy and Pydantic).
- **SQLite:** The database where all the info is stored.
- **HTML & CSS:** For the layout and styling.
- **JavaScript:** For some simple frontend logic.

### How to run it
1. Make sure you have Python installed.
2. Install the requirements using `pip install flask sqlmodel pydantic`.
3. Run the project with `python main.py`.
4. Open your browser and go to `http://localhost:8888`.
